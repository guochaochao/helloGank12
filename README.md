# helloGank
### 一个gank Android客户端
### 写了一个gank Android客户端，项目地址：https://github.com/guochaochao/helloGank12
### 主要实现下面功能：
* 借鉴简书实现夜间模式切换，没有使用框架
* 实现视屏播放，功能包括全屏切换，手势控制音量和亮度
* 音乐播放正在开发中
* 查看妹子图片放大缩小功能
